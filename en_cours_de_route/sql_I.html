SQL Avancée | Procédure stockée

Les procédures stockée permet d'enregistrée une commande sql par le biais d'une fonction, fonction possible car phpMyAdmin reste une interface faite de php. Ces procédures executée sont enregirstrées dans le systems pour les voir ils suffit de faire SHOW PROCEDURE STATUS
L'avantage de la procédure est qu'elle est réeutilisable plus tard avec CALL suivit du nom de la fonction(Prenant un arguments). Plusieurs paramètre peuvent être entrez dans la fonction, toutefois à l'appel de la procédure, cela demanderas autant d'argument que de paramètres choisie initialement.
La requète préparée peu reçevoir autre chose qu'un SELECT, elle peu reçevoir toutes les commandes courrantes pour la gestions d'une base de donnée, de table de cette dernières ou de données.

DELIMITER |
CREATE PROCEDURE liste_quelconque(In title Varchar(100))
BEGIN
SELECT pro_id,pro_title,pro_price FROM produits
WHERE pro_title = title;
END |
DELIMITER ;
CALL liste_quelconque('FF-4000');
